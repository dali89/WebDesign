BackEnd
The back-end is the code that runs on the server, that receives requests from the clients, and contains the logic to send the appropriate data back to the client. The back-end also includes the database, which will persistently store all of the data for the application.
HTML, or HyperText Markup Language, is a programming language that instructs web browsers on how to structure and display web pages.
Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML.
Bootstrap is a free, open-source front-end framework that helps developers build responsive websites faster. It's built on HTML, CSS, and JavaScript, and includes templates for typography, buttons, forms, navigation, and more.
JavaScript is a scripting language that allows developers to create interactive web pages that can interact with visitors and perform complex actions.
